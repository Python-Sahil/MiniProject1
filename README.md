# Microproject #1 - Express + Node Website

## Description
This is a simple web application built with Node.js and Express. It serves static HTML pages and provides an API endpoint (`/api/items`) that returns JSON-data from a local file (`data/items.json`). The front-end demonstrates fetching that data and rendering it dynamically.

## Technologies
- Node.js
- Express
- HTML/CSS/JavaScript
- Async/Await fetch API
